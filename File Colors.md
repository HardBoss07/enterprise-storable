# File Colors

| File Type           | Extensions (Examples)                        | Component Name     | Recommended Color              |
| ------------------- | -------------------------------------------- | ------------------ | ------------------------------ |
| **Stylized Text**   | `docx`, `rtf`, `odt`, `pages`                | `FileText`         | **Word Blue** (`#2b579a`)      |
| **Fully Fletched**  | `pdf`                                        | `FileText`         | **Adobe Red** (`#f40f02`)      |
| **Slideshows**      | `pptx`, `odp`, `key`                         | `FileChartColumn`  | **PPT Orange** (`#d24726`)     |
| **Spreadsheets**    | `xlsx`, `ods`, `csv`                         | `FileSpreadsheet`  | **Excel Green** (`#217346`)    |
| **Images**          | `png`, `svg`, `jpeg`, `jpg`, `webp`          | `FileImage`        | **Sky Blue** (`#00a4ef`)       |
| **Archives**        | `rar`, `zip`, `gzip`, `jar`, `7z`            | `FileArchive`      | **Folder Yellow** (`#ffb900`)  |
| **Audio Files**     | `mp3`, `wav`, `m4a`, `flac`                  | `FileVolume`       | **Vibrant Pink** (`#ec4899`)   |
| **Video Files**     | `mp4`, `mov`, `webm`, `mkv`                  | `FileVideoCamera`  | **Royal Purple** (`#8b5cf6`)   |
| **3D Files**        | `obj`, `fbx`, `stl`, `glb`                   | `FileBox`          | **Indigo** (`#6366f1`)         |
| **Config Files**    | `conf`, `gitignore`, `env`, `ini`            | `FileCog`          | **Slate Gray** (`#64748b`)     |
| **Markup Lang**     | `html`, `xml`, `yml`, `yaml`                 | `FileCodeCorner`   | **Deep Orange** (`#E44D26`)    |
| **Source Code**     | `rs`, `ts`, `jsx`, `c`, `php`, `sql`, `java` | `FileBracesCorner` | **Teal / Emerald** (`#10b981`) |
| **Key / Secured**   | `asc`, `pgp`, `gpg`, `pem`, `pub`            | `FileKey`          | **Goldenrod** (`#d4af37`)      |
| **Text Fallback**   | `txt`, `log`, `md`                           | `FileText`         | **Cool Gray** (`#94a3b8`)      |
| **Binary Fallback** | `bin`, `exe`, `dll`, `class`                 | `File`             | **Charcoal** (`#475569`)       |
